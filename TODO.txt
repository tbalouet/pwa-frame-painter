     //Mise en place firebase
     Pour tester sur une prod sécurisée, le plus simple est de passer par firebase.
     1) Créer le projet sur https://console.firebase.google.com
     2) Cliquer sur "hosting" dans la page d'accueil du projet
     3) Dans la console windows firebase login --interactive (lance la fenêtre d'authentification google)
     4) firebase init (sélectionner hosting et suivre les indications)
     5) Eventuellement modifier le firebase.json pour écarter des dossiers (par exemple node_modules):
     {
        "hosting": {
          "public": "./",
          "ignore": [
            "firebase.json",
            "**/.*",
            "**/node_modules/**"
          ],
          "rewrites": [
            {
              "source": "**",
              "destination": "/index.html"
            }
          ]
        }
      }
      6) Firebase deploy
      7) Accéder sur PC l'adresse donnée dans la console
      8) Avec le plugin The QR Code Extension dans Google Chrome, générer le QR code de la page
      9) Sur mobile, avec le navigateur Samsung Internet, scanner le QR code et checker la pwa
